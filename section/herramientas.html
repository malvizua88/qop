Pulverizador

Es muy eficaz para dotar de humedad a las plantas y mantener las hojas limpias de polvo. Se puede utilizar, también, para humedecer la tierra de los pequeños semilleros.

Escarificador de mano

El escarificador de mano sirve tanto para descortezar y mullir la tierra como para arrancar malas hierbas.

Regadera

Se puede utilizar tanto fuera como dentro del hogar. Hay diferentes tipos de regadera en función de las plantas en las que se vaya a usar: de cuello largo, corto...

Transplantador y tridente

El trasplantador es una pequeña pala que se utiliza para trasplantar en las jardineras y macetas. El tridente sirve para soltar y mullir la tierra alrededor de las plantas.

Tijera de mano y de podar

Este tipo de tijera es perfecta para podar todas vuestras plantas de interior. Muchos nos soléis preguntar cuándo se debe hacer y, en realidad, todo depende del fin que se quiera conseguir.
